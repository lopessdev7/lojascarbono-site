# 🌐 Projeto Web - Site Institucional

Este projeto é um site institucional desenvolvido sob medida para um cliente, com foco em presença digital, conversão e apresentação profissional do negócio.

---

## 🧠 Visão do Projeto

O objetivo foi criar uma página moderna, leve e responsiva, capaz de transmitir credibilidade e facilitar o contato com o cliente final.

---

## ⚙️ Solução Entregue

- Desenvolvimento de site institucional responsivo  
- Estrutura otimizada para mobile e desktop  
- Design focado em clareza e conversão  
- Publicação e hospedagem em ambiente online  

---

## 🛠️ Tecnologias Utilizadas

- HTML5  
- CSS3  
- JavaScript  
- Deploy via Vercel  

---

## 🎯 Foco do Projeto

- Presença digital profissional  
- Geração de autoridade para o negócio
